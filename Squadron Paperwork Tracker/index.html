Welcome to our website!

Here to make life easier...

All of the various forms that are used within the squadron do not have a set place to store them nor is there a way to sort through to find how many people earn positive and negative paperwork at the academy in order to organize the information. With the Squadron Paperwork Tracker, people can easily store forms in logical places and access them when needed. This database will facilitate the jobs within squadrons, such as element leaders who need to input feedback forms for multiple people.

Functions that the user will be able to perform on data will be displayed in an interactive GUI. These functions will include, but are not limited to, file form, lookup form, and delete form. The functions should be as easy to use - you only need to click what you want and then what you want to do with it.
